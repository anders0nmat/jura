# Deutsche Gesetzbücher

Schnell durch Gesetze navigieren ohne Schnickschnack

## Features

Seite verfügbar unter https://anders0nmat.github.io/jura/.

Suche funktioniert einfach durch lostippen.

Suche nach Gesetzbüchern oder Paragraphen/Artikeln.

Navigation:
- Hoch/Runter innerhalb von Suchergebnissen
- Enter um Suchergebnis zu öffnen
- Links/Rechts innerhalb von Paragraphen
- STRG+Links/Rechts innerhalb der angepinnten Paragraphen
- Enter um Paragraph anzupinnen/loszulösen

## Warum?

1. https://gesetze-im-internet.de hatte mir nicht genug Features.
2. Gesetze sollte man ohne Werbung nachschlagen können. Meine Meinung.

## Sonstiges

Datenquelle: https://gadi.netlify.app

Gesetze werden bein ersten Abrufen für eine Woche lokal gespeichert. Ermöglicht schnelleres Abfragen und entlastet Datenquelle.
